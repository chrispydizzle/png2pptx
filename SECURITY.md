# Security Policy

## Supported status

`png2pptx` is currently a **v0.1 beta** local CLI tool. It is best treated as a workstation utility, not a hardened service for hostile or untrusted multi-tenant environments.

## Reporting a vulnerability

Please do **not** post full exploit details in a public issue.

If GitHub private vulnerability reporting is enabled for the repository, use that channel. Otherwise, open a minimal issue asking for a private contact path without including sensitive details.

## What to include

Please include:

- affected version or commit
- operating system
- reproduction steps
- impact
- any suggested mitigation

## Response expectations

Because this is a beta project, response times are best-effort rather than guaranteed.
